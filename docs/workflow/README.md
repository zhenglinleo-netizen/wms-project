# WMS系统工作流程文档

## 文档目录

### 1. 核心模块功能分析
- **文件**：`CORE_MODULES_ANALYSIS.md`
- **内容**：详细分析系统各个核心模块的功能、输入输出和内部工作流程
- **适用人群**：开发人员、运维人员、项目经理

### 2. 模块联动关系分析
- **文件**：`MODULE_INTERACTIONS.md`
- **内容**：分析模块间的调用关系、数据流向和事件触发机制，包含模块联动关系图
- **适用人群**：开发人员、运维人员

### 3. 系统工作流程分析
- **文件**：`SYSTEM_WORKFLOW.md`
- **内容**：系统整体工作流程、各个核心模块的工作流程图、系统核心业务流程（可将代码粘贴复制到DrawIO在线绘图网站中查看流程图）
- **适用人群**：运维人员、项目经理、系统管理员

## 文档用途

这些文档旨在帮助运维人员理解WMS系统的工作流程和模块间的联动关系，便于系统的维护和故障排查。文档包含了详细的工作流程图和文字说明，覆盖了系统的所有核心模块和业务流程。

## 如何使用这些文档

1. **系统架构理解**：首先阅读`CORE_MODULES_ANALYSIS.md`，了解系统的核心模块和它们的功能

2. **模块联动关系**：然后阅读`MODULE_INTERACTIONS.md`，了解模块间的联动关系和数据流向

3. **工作流程掌握**：最后阅读`SYSTEM_WORKFLOW.md`，掌握系统的整体工作流程和核心业务流程

4. **故障排查参考**：当系统出现问题时，参考这些文档，了解模块间的联动关系，快速定位问题所在

## 文档更新

当系统发生变更时，应及时更新这些文档，确保文档与系统的实际实现保持一致。

## 技术支持

如果在使用这些文档时遇到问题，请联系系统开发团队获取技术支持。
